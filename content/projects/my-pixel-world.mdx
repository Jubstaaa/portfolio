# How It Works: Gameplay Overview

In this collaborative pixel art editor, your goal is simple:  
Create beautiful pixel art together with other users in real-time.

Here's how the application works:

- **Choose Your Tool:** Select from pen, eraser, or hand tools for different drawing needs.
- **Real-Time Collaboration:** Draw simultaneously with other users — see their pixels appear instantly.
- **Room-Based System:** Join different rooms for focused collaboration or create custom rooms.
- **Large Canvas:** Work on a spacious 2000x2000 pixel canvas with smooth panning and zooming.
- **Color Selection:** Use the intuitive color picker to choose from millions of colors.
- **Live Synchronization:** All changes are instantly shared with connected users.

# Multiple Features & Expanding Capabilities

**My Pixel World** features a comprehensive set of drawing tools and collaborative features, including:

- **Real-Time Drawing:** See other users' pixels appear instantly as they draw
- **Advanced Tools:** Pen tool for precise pixel placement, eraser for corrections, and hand tool for navigation
- **Color Picker:** Full RGB color spectrum with intuitive selection
- **Large Workspace:** Spacious 2000x2000 pixel canvas for detailed artwork
- **Smooth Navigation:** Pan around the canvas with the hand tool for easy access to all areas
- **Room System:** Multiple collaborative rooms with dynamic navigation and popular rooms feature
- **Connection Status:** Real-time indicator showing connection status to the server
- **Popular Rooms:** Real-time sorting of rooms based on pixel count and activity

Each feature is designed to enhance the collaborative drawing experience, and new capabilities are added regularly.

# Technical & Performance Features

- **Real-Time Communication:** WebSocket-based communication ensures instant pixel synchronization across all connected users.
- **Memory-Based Architecture:** Lightning-fast performance with all operations in memory, backed by MongoDB for data persistence.
- **Data Persistence:** Drawing history is automatically saved to MongoDB with memory-based architecture for optimal performance.
- **Room Management: **Efficient room system with automatic creation and popular rooms calculation.
- **Scalable Architecture:** Built with Next.js frontend and Node.js backend, deployed on Vercel (frontend) and Digitalocean.com (backend with MongoDB) for optimal performance.
