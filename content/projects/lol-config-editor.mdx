### What Is It?

LoL Config Editor is a lightweight desktop app that lets you create named profiles for every playstyle, account, or roommate, then activate any profile with one click. Think of it as your personal “loadout switch” for League settings.

---

### How It Works • Zero Friction Pipeline

1. **Profile-First Design**
2. - Craft a setup (resolution, keybinds, chat state, minimap size, color-blind mode, etc.).

- Save it under any name—_Ranked_, _ARAM_, _Guest_.

3. **One-Click Apply**
4. - Select a profile → hit **Apply** → launch League.

- The chosen config propagates to every LoL account on that PC.

5. **Multi-User Smartness**
6. - Each person on a shared computer keeps their own library of configs.

- No more “Who messed up my hotkeys?”

### Headline Features

- **Game-Free Editing** – Adjust everything outside the client.
